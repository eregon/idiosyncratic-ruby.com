Misc fun stuff

How to produce this output? (1)


[1,2].____ { |x| [x, x**2] }
# => {1 => 1, 2 => 4}

How to produce this output? (1)


[1,2].to_h { |x| [x, x**2] }
# => {1 => 1, 2 => 4}


Before, we had to use Hash[]

How to produce this output? (2)


[*(1..20)_3]
#=> [1, 4, 7, 10, 13, 16, 19]

How to produce this output? (2)


[*(1..20)%3]
#=> [1, 4, 7, 10, 13, 16, 19]


The new Range#% is an alias to Range#step

How to produce this output? (3)


a = (7..9).each

How to produce this output? (3)


a = (7..9).each
b = Prime.each
a.chain(b).take(7)
#=> [7, 8, 9, 2, 3, 5, 7]
