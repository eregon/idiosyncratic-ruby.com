Question 3

There is a String#split method,
which also accepts regexes:


"rug-b".split(/-/) #=> ["rug", "b"]

What will happen if we use a capturing group in the regex?


  "rug-b".split(/(-)/) #=> ?

Question 3


"rug-b".split(/(-)/) #=> ["rug", "-", "b"]

Bonus Question 3

What will happen if we use a multiple capturing groups?


"rug-b".split(/((-))/) #=> ?

Bonus Question 3

What will happen if we use a multiple capturing groups?


"rug-b".split(/((-))/) #=> ["rug", "-", "-", "b"]

Bonus Bonus


"rug-b".split(/(((((-)))))/)
#=> ["rug", "-", "-", "-", "-", "-", "b"]
            

"rug-b".split(/-(?=(.))/)
#=> ["rug", "b", "b"]
